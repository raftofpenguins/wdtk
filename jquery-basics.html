Getting Started with jQuery

  • After linking your .js file to your HTML document as detailed here, you will also need to include the following link to the jQuery library in the Head of your HTML document:

    <script src="https://code.jquery.com/jquery-1.9.1.min.js">
    </script>
    • INCLUDE THIS LINK ABOVE THE LINK TO THE EXTERNAL .js FILE!!!
  • In your .js file, you typically create a jQuery function using the following syntax:

    $(document).ready(function() {

    });
  • Within this jQuery function, you can create events that interact with HTML elements like so:

    $(document).ready(function() {
        $("#htmlElement").anyAction(arg);
        $("#htmlElementA, #htmlElementB").anotherAction();
    });
    • Where you see "#htmlElement", you can target any HTML element as you would with a CSS selector.
      • You can also include any number of elements by separating them with commas (i.e. use a "compound selector").
    • You can replace "anyAction(arg)" with one of jQuery's many events or event handlers.
  • Create variables that contain numbers, strings, or even HTML elements that are now jQuery objects:

    var $varNum = 5;
    var $varString = "Words and stuff";
    var $varElement = $("htmlElement");
  • Use the "this" keyword to limit actions to only affecting the element with which you interacted:

    $(document).ready(function() {
        $('div').click(function() {
            $(this).fadeOut('slow');
        });
    });
    • While the above "click" function will potentially affect any clicked div, it only fades out the actual div that was clicked.

Creating, Moving and Removing HTML Elements with jQuery

  • Use the "append" and "prepend" functions to create new child HTML elements within a parent element:

    $(document).ready(function(){
        $("body").append("<p>I'm a paragraph!</p>"); 
    });
    • Creates a paragraph as the last child element of the body; using "prepend" would make it the first child.
    • Can also use "appendTo" and "prependTo", which simply reverse the target and addition places.
  • Use the "before" and "after" functions to create new HTML elements before or after another element:

    $(document).ready(function() {
        $("#htmlElement").after("<p>Penguin paragraph!</p>")
    });
    • Creates a paragraph after the target element.
  • Use the same functions as above with existing elements rather than creating new ones in order to rearrange elements:

    $(document).ready(function() {
        $("#one").after("<p>Penguin Paragraph!</p>");
        var $penguinPara = $("p");
        $("#two").after($penguinPara);
    });
    • Use of a variable such as the $penguinPara shown above facilitates moving a single element rather than creating a copy.
  • The "empty" and "remove" functions help to delete HTML elements:

    $(document).ready(function() {
        $("#one").after("<p>Penguin Paragraph!</p>");
        var $penguinPara = $("p");
        $("#two").after($penguinPara);
        $penguinPara.remove();
    });
    • "Remove" will remove the element entirely
    • "Empty" will empty the contents of the element and all its descendants.

Modifying Classes and Content with jQuery

  • Use the "addClass", "removeClass" and "toggleClass" functions to add, remove or toggle classes from an element:

    $(document).ready(function() {
        $("#targetElement").click(function() {
            $("#targetElement").addClass("newClass");
        });
    });
    • Gives the target element a new class when it is clicked.
    • "removeClass" and "toggleClass" have similar syntax as the above example.
  • Use the "height" and "width" functions to change the height and width of an element:

    $(document).ready(function() {
        $("div").height("80px");
        $("div").width("90px");
    });
    • Gives all divs a height of 80 and a width of 90.
  • Use the "css" function to change any property of an element:

    $(document).ready(function() {
        $("div").css("height", "80px");
    });
    • Gives all divs a height of 80, similar to the height function.
  • Use the "html" function to change the contents of an element:

    $(document).ready(function() {
        $("div").html("Penguins are fun!");
    });
    • Changes the contents of the first div found to "Penguins are fun!".
  • Use the "val" function to change the contents of a form element.

    $(document).ready(function() {
        var input = $('input[name=checkListItem]').val();
    });
    • Stores value of checklist item in variable "input".
