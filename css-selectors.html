What Are Selectors?

Selectors allow you to target and style specific elements on a webpage.

  • Types of Selectors:
    • HTML Selectors
    • ID and Class Selectors
    • Pseudo-Class Selectors

HTML Selectors

HTML Selectors allow CSS to apply styling to specific HTML elements.

  • All HTML elements are Selectors.
  • E.g., the following code selects a page's paragraphs:

    p {
        
    }

ID and Class Selectors

ID and Class selectors allow an extra degree of specificity when choosing which elements to style.

  • IDs can be used to style one specific element by putting a # in front of the ID.
  • E.g., to style an introductory paragraph with the ID "introPara", we could use the following selector:

    #introPara {
        
    }
  • Classes can be used to style several similar elements by putting a . in front of the Class name.
  • E.g., to style body paragraph with the class "bodyPara", we could use the following selector:

    .bodyPara {
        
    }
  • Putting it all together, we will need to specify which paragraphs get what styling type in the HTML file.
  • Here is an example of applying the introductory and body paragraph stylings:

    <p id="introPara">This is an intro paragraph.</p>
    <p class="bodyPara">This is a body paragraph.</p>
    <p class="bodyPara">This is another body paragraph.</p>

Pseudo-Class Selectors

Pseudo-Class Selectors are used for changing different aspects of an element at different times, such as when it is clicked.

  • Use a colon and pseudo-class selector to begin styling various states of an element.
  • E.g., the following code selects a page's links when the mouse hovers over them:

    a:hover {
        
    }
  • More examples of when styling is applied:

    element:hover { }           /* Mouse hovers over element */
    a:link { }                  /* Unvisited link */
    a:visited { }               /* Visited link */
    element:first-child {}      /* First element only */
    element:nth-child(#) {}     /* Nth element only (insert #) */
