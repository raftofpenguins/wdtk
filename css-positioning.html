Positioning in CSS

Mastering positioning properties in CSS will help everything in your website fall right into place!

  • The following topics will help you get started with positioning in CSS:
    • The Box Model
    • Positioning Properties
    • Floating Elements
    • Absolute, Relative and Fixed Positioning

The Box Model Diagram

The w3schools.com website describes the Box Model as being a way of describing how HTML elements sit on a page.

  • Most elements sit on a webpage as a block by default, taking up the full width of the page.
  • Content sits in the middle of the block with several layers around it.

Positioning Properties

  • Display Properties affect how HTML elements sit next to each other on the page.


    anyElement { 
        display: block;             /* The element takes up full width of 
                                        page; nothing sits next to it */

        display: inline-block;      /* The element is a box but other 
                                        elements can sit next to it on 
                                        same line */

        display: inline;            /* The element sits on same line as 
                                        other elements without formatting 
                                        it like a block */

        display: none;              /* The element and its content 
                                        disappears from page entirely */
    }

  • Spacing Properties affect the margins, border, and padding of elements.


    anyElement { 
        margin: 20px;                   /* Keeps 20 pixels between element's 
                                            borders and other elements */

        margin: 10px, 15px, 20px, 25px; /* Specify each side's margin in
                                            clockwise order: top, right,
                                            bottom, left */

        margin-left: 20%;               /* Keeps left side of the element's
                                            margin 20% of the page's width 
                                            away from nearest thing on left */
    }
    • An element's margin property affects the distance between the element's border and other elements.
    • There are multiple ways of selecting certain sides of the element and specifying distances, as seen in the above examples.

    anyElement { 
        padding: 5px;                   /* Keeps 5 pixels between element's 
                                            content and its border */
    }
    • An element's padding property is similar to the margin property, but affects the space between an element's border and its content.

    anyElement { 
        border: 
            2px                     /* Width of border; set to 0 for no border */
            solid                   /* Style of line, e.g. solid, dashed */
            black;                  /* Color of line; use words or hex codes */

        border: 2px solid black;    /* More common way of formatting 
                                        the border property */
    }
    • An element's border property affects the style of the line surrounding an element.

Floating Elements

  • The float property facilitates the placement of multiple elements in particular locations on the page.


    anyElement { 
        float: right;               /* Makes the element sit to the right */
        float: left;                /* Makes the element sit to the left */
    }

  • The clear property moves an element below others if they would otherwise be on the same line.


    anyElement { 
        clear: right;               /* Makes the element clear elements 
                                        on the right */

        clear: left;                /* Makes the element clear elements 
                                        on the left */

        clear: both;                /* Makes the element clear elements 
                                        on both sides */
    }

Absolute, Relative and Fixed Positioning

  • The position property can position an element relative to another, or fixed in the browser window.


    anyElement { 
        position: static;           /* Default position */

        position: absolute;         /* Positioned relative to first 
                                        non-absolute parent */

        position: relative;         /* Position relative to where the 
                                        element would sit by default (static) */

        position: fixed;           /* Positions the element in a fixed
                                        area of the browser window */
    }
